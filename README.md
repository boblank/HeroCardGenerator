# 人物卡牌生成器
利用`PhantomJS`来生成三国杀风格的人物卡牌。

![](./samples/screenshot.png)

## 使用说明
1. [安装PhantomJS](http://phantomjs.org/)
2. 修改`index.html`和`card.css`, 完成卡牌设计
3. `phantomjs print.js` 生成png格式图片

> 已包含可商用中文字体
